# اسكربتات إدراج جميع المنتجات

هذا الملف يحتوي على جميع الأوامر اللازمة لإدراج المنتجات في قاعدة البيانات.

## الترتيب الصحيح لتشغيل السكريبتات

يجب تشغيل السكريبتات بالترتيب التالي من مجلد `backend`:

### 1. إضافة المواد الكيميائية (361 منتج)
```bash
npx tsx src/scripts/addChemicals.ts
```

### 2. إضافة المنظفات ومستحضرات التجميل (33 منتج)
```bash
npx tsx src/scripts/addDetergents.ts
```

### 3. إضافة الزجاجيات المخبرية (21 منتج)
```bash
npx tsx src/scripts/addGlassware.ts
```

### 4. إضافة المزيد من الزجاجيات (28 منتج)
```bash
npx tsx src/scripts/addMoreGlassware.ts
```

### 5. إضافة الأجهزة المخبرية (22 منتج)
```bash
npx tsx src/scripts/addLabEquipment.ts
```

### 6. إضافة المحاليل والكواشف (71 منتج)
```bash
npx tsx src/scripts/addReagents.ts
```

### 7. إضافة الأوساط الزراعية والمضادات الحيوية (70 منتج)
```bash
npx tsx src/scripts/addProducts.ts
```

## أمر واحد لتشغيل جميع السكريبتات

يمكنك تشغيل جميع السكريبتات دفعة واحدة باستخدام الأمر التالي:

```bash
npx tsx src/scripts/addChemicals.ts && npx tsx src/scripts/addDetergents.ts && npx tsx src/scripts/addGlassware.ts && npx tsx src/scripts/addMoreGlassware.ts && npx tsx src/scripts/addLabEquipment.ts && npx tsx src/scripts/addReagents.ts && npx tsx src/scripts/addProducts.ts
```

## التحقق من البيانات

بعد إدراج المنتجات، يمكنك التحقق من عددها باستخدام:

```bash
npx tsx src/scripts/checkData.ts
```

## ملاحظات مهمة

1. تأكد من أن قاعدة البيانات PostgreSQL تعمل قبل تشغيل السكريبتات
2. تأكد من وجود ملف `.env` في مجلد `backend` مع إعدادات قاعدة البيانات الصحيحة
3. السكريبتات تتحقق من وجود الأقسام والأقسام الفرعية وتنشئها تلقائياً إذا لم تكن موجودة
4. إذا كانت المنتجات موجودة مسبقاً، قد تحصل على أخطاء duplicate، وهذا طبيعي

---

## في حالة فقدان البيانات

إذا فقدت البيانات في أي وقت:

1. افتح Terminal في مجلد `backend`
2. شغل الأمر الشامل أعلاه
3. انتظر حتى تنتهي جميع السكريبتات
4. تحقق من البيانات باستخدام `npx tsx src/scripts/checkData.ts`
